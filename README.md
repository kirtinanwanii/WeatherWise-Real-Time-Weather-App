# WeatherWise – Real-Time Weather App

WeatherWise is a responsive and interactive weather application built using **HTML**, **CSS**, and **JavaScript**.  
It fetches live weather data from a public API and displays it in a clean, user-friendly interface.

---

## Live Demo
Try the app online:  [Live Demo](https://kirtinanwanii.github.io/WeatherWise-Real-Time-Weather-App/)  

---

##  Features

-  Check real-time weather for any city  
-  Display temperature, humidity, wind speed, and conditions  
-  Clean, responsive UI  
-  API integration using JavaScript fetch  
-  Beginner-friendly code structure  

---

##  Technologies Used

- **HTML** – App structure  
- **CSS** – Styling and layout  
- **JavaScript** – Fetch API calls, DOM updates, interactivity
